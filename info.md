# Slim Gazon Sproeien

Weerafhankelijk, slim sproeien voor je gazon. Berekent elke ochtend een dagplan
op basis van temperatuur, grasfase, regen, wind, zon, UV en luchtvochtigheid, en
voert de beurten veilig uit met je grote en kleine sproeier.

- Volledig instelbaar via de UI en via entiteiten (geschikt voor een dashboard).
- Stopt bij regen en bewaakt de maximale looptijd.
- Start standaard in **testmodus** (dry-run) zodat je veilig kunt afstellen.

Na installatie: **Instellingen → Apparaten & diensten → Integratie toevoegen →
Slim Gazon Sproeien**.
